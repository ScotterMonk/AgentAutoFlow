### Scoring rules

| Rank | Brand & Product Name | Factors | Key Findings & Analysis |
| :--- | :--- | :--- | :--- |
| 1 | **Purity (Ingredients / “what is it, exactly?”)** | Score **5→1** rubric | Purity is an ingredient-deck test. **5** = `cream` only (unsalted) or `cream + salt` (salted). **3** = extra dairy “fillers” like **skim milk/milk solids**. **1–2** = flavorings, gums, stabilizers, “spreadable butter” type formulations, etc. |
| 2 | **Sourcing/Omega (feed quality proxy)** | Score **5→1** rubric | This factor rewards “how the cows were fed” (as a proxy for fatty-acid profile). **5** = explicitly **100% grass-fed/grass-finished** (or equivalent strongest pasture claim). **4** = organic / pasture-forward but not clearly 100% grass-fed. **3** = “pasture-raised” *but* meaningful supplemental feed is a drawback (ex: Vital Farms). |
| 3 | **Seed oils (added vegetable oils)** | Score **5→1** rubric |This is a pass/fail-style category. **5** = no added vegetable/seed oils. **Lower scores** would apply to “spreadable butter” products that add vegetable oils to soften at fridge temps (a known mfg approach). |
| 4 | **Emulsifiers (lecithin, mono/diglycerides, gums, etc.)** | Score **5→1** rubric | Also pass/fail in your examples. **5** = no emulsifiers/stabilizers. **Lower scores** if label includes lecithin, mono-/diglycerides, polysorbates, gums, etc. This is consistent with the fact that emulsifiers can be used to alter butter’s properties/processing behavior. |
| 5 | **Processing (raw vs pasteurized; cultured vs sweet cream; “industrial-ness”)** | Score **5→1** rubric | **5** = **raw/unpasteurized**. **4** = pasteurized but **cultured/vat-cultured**  **2** = standard **sweet cream** butter (fresh pasteurized cream, not fermented). Sweet cream butter is typically pasteurized cream w/o fermentation. |
