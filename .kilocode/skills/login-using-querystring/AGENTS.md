# Skill-local agent guidance: login-using-querystring

This project is a Tkinter/CLI sync utility and does not currently expose a web login flow.

## Project guidance
- Do not assume a local Flask login route exists for this repository.
- If a future web UI adds querystring login, document the local URL, host restrictions, and credential lookup here.
- Use `browser-use` only for relevant web UI or static HTML verification tasks.
