# Skill-local agent guidance: database-use

This project has no database.

## Project guidance
- Do not add database setup, migrations, schema inspection, seed data, or DB teardown for this application.
- If a task appears to require persistence, first inspect the existing file-based configuration and sync behavior.
- If a database is explicitly introduced later, document its schema, migration workflow, credentials location, utility modules, and tests here instead of putting those details in `SKILL.md`.
