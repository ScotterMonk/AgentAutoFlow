# Skill-local agent guidance: browser-use

## Project browser context
- This repository is primarily a Tkinter/CLI sync utility, not a web app.
- If a local web surface is added or being tested, use the project command from root guidance, typically `py app.py` only when applicable.
- Bundled browser helper scripts live under `.kilocode/skills/browser-use/scripts/`.
- Bundled examples live under `.kilocode/skills/browser-use/examples/`.
- Prefer relative `outputs/` paths for screenshots and logs.
